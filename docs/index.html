The pipeline

01

Synthesize

Generate images from a prompt — hosted Gemini (Nano Banana) or a deterministic mock that runs anywhere.

02

Auto-label

GroundingDINO proposes boxes; refine them in Label Studio and pull clean COCO back out.

03

Train

A tiny soft-argmax CNN — five stride-2 conv blocks — quantized down to an int8 ONNX model.

04

Run

onnxruntime on CPU, driven from a CLI or exposed over MCP. An edge-sized inference loop.

The detection contract

Four bytes per detection. At 480 px the quantization step is ~1.88 px, so round-trip error stays under 1 px and all post-processing lives in 8-bit math. 

# run a model on a frame, from the repo root
uv run tensor-factory detect --model model.onnx --image frame.png

# or in Python
from tensor_factory import BBox, encode_uint8
encode_uint8(BBox(0.1, 0.1, 0.9, 0.9))   # (26, 26, 230, 230)

Five packages

tensor-factory

Core: BBox geometry, the 4×uint8 codec, annotation formats, ONNX inference + CLI. Dependency-free, CPU-only.

tensor-factory-synth

Generation (Nano Banana / Gemini) plus GroundingDINO auto-label and COCO / Label Studio export.

tensor-factory-train

The tiny soft-argmax detector, exported to int8 ONNX.

tensor-factory-mcp

FastMCP server exposing the detector as tools, with a bundled demo model.

tensor-factory-label

Label Studio integration: push candidates, pull corrected labels.
